# Lilith OS

该项目旨在通过学习Rust语言和操作系统原理，实现一个基本的操作系统。

- lilith OS 是一个基于 Rust 语言构建的轻量级操作系统，致力于达成简单、高效与良好兼容性的目标。
- lilith-kernel 是基于 Rust 语言的操作系统内核，旨在为系统运行提供简洁且高效的环境内核。

## 测试

```bash
cargo test
```

## 快速开始

```bash
# 编译
cargo build

# 运行
cargo run

# 完整构建
make build

# QEMU启动测试
make run
```


## 功能实现路径

1. **基础内核功能**
   - ✅ **内核启动**  
     实现内核启动过程，确保系统能正确加载内核镜像。
   - ✅ **内核初始化**  
     完成基本子系统的初始化工作，为后续功能提供支持。
   - 🚧 **中断处理**  
     - 细化中断向量表配置  
     - 构建中断分发机制，实现优先级管理及中断嵌套处理
   - 🚧 **异常处理**  
     - 定义异常检测与响应策略  
     - 实现异常捕获、记录及安全退出流程

2. **引导加载及调试支持**
   - 🚧 **引导加载器（Multiboot2规范）**  
     - 完善Bootloader与内核加载流程  
     - 增加自检与硬件初始化步骤
   - 🚧 **基础串口调试输出**  
     - 配置串口通信参数（波特率、数据位、停止位）  
     - 实现调试信息输出模块，便于内核调试

3. **内存管理功能**
   - 🚧 **物理内存管理（Buddy分配器）**  
     - 完成内存块划分、合并及回收算法  
     - 添加内存使用情况的监控和日志记录
   - 🚧 **虚拟内存管理（分页机制）**  
     - 初始化页表并实现内存映射  
     - 支持地址转换及缺页异常处理
   - 🚧 **内核堆内存分配（slab分配器）**  
     - 实现内核动态内存分配与释放  
     - 添加内存碎片管理和性能测试

4. **中断控制及调度**
   - 🚧 **硬件中断控制器（APIC）**  
     - 配置多核中断分发  
     - 实现中断响应及处理流程
   - 🚧 **进程调度器（Round-Robin）**  
     - 完成基本调度算法实现  
     - 集成调度与上下文切换测试，优化时间片分配
   - 🚧 **系统调用接口**  
     - 定义系统调用入口及调度机制  
     - 实现用户态与内核态间的安全交互

5. **用户与设备支持**
   - 🚧 **用户态进程加载器**  
     - 搭建用户程序运行环境  
     - 实现加载、错误处理及资源分配
   - 🚧 **虚拟文件系统（VFS抽象层）**  
     - 定义文件系统接口  
     - 实现文件的挂载、读写及缓存机制
   - 🚧 **基础块设备驱动**  
     - 实现对存储设备的低级读写支持  
     - 增加设备错误处理与恢复机制
   - 🚧 **EXT2文件系统支持**  
     - 完成EXT2挂载、读取及写入功能  
     - 添加一致性校验和异常恢复
   - 🚧 **TTY终端子系统**  
     - 配置终端输入输出处理  
     - 提供基础命令行交互接口
   - 🚧 **用户空间库（libc基础）**  
     - 实现常用C标准库函数  
     - 确保兼容用户程序的调用需求

6. **模块化微内核架构及核心服务**
   - 🚧 **核心服务分离（进程管理、内存管理、进程间通信）**  
     - 设计各核心服务的接口与整体架构  
     - 实现基本功能，并进行单元测试  
     - 集成进程管理、内存管理和进程间通信模块，完成系统级测试
   - 🚧 **基于消息传递的进程间通信（IPC）机制**  
     - 定义消息格式和通信协议  
     - 实现IPC数据传输及错误检测
   - 🚧 **动态模块加载器（ELF格式内核模块支持）**  
     - 实现内核模块的动态加载与卸载  
     - 处理模块依赖及安全校验
   - 🚧 **运行时模块热插拔配置系统**  
     - 实现模块无缝加载、替换与卸载  
     - 构建配置管理接口，实现实时更新
   - 🚧 **微内核验证测试框架（基于QEMU）**  
     - 搭建QEMU测试环境  
     - 编写测试用例，验证微内核各项功能的稳定性

7. **高级功能与扩展**
   - 🚧 **进程管理及内存管理优化**  
     - 深入优化进程调度算法和内存分配策略  
     - 通过性能测试进一步提升系统效率
   - 🚧 **ACPI电源管理**  
     - 实现电源状态监控与系统休眠、唤醒管理

   - 🚧 **QEMU集成测试**

     - 修正测试中发现的兼容性与性能问题
   - 🚧 **异步驱动框架**

   - 🚧 **网络协议栈**  
     - 实现TCP/IP基本通信协议  
     - 支持数据包收发及简单网络服务功能
   - 🚧 **GUI图形界面**  
     - 设计基本界面布局与窗口管理  
     - 实现基础控件（按钮、文本框等）及事件处理
   - 🚧 **多格式文件支持**


## lilith文件结构

```
/
├── bin/        # 基础命令二进制文件
├── boot/       # 系统引导文件（内核、GRUB）
├── dev/        # 设备文件接口（磁盘、终端等）
├── etc/        # 系统配置文件（passwd、ssh等）
├── home/       # 普通用户主目录
├── lib/        # 核心共享库（.so文件）
├── lost+found/ # 系统崩溃恢复文件
├── media/      # 自动挂载移动设备
├── mnt/        # 临时手动挂载点
├── opt/        # 第三方软件安装目录（如Chrome）
├── proc/       # 进程/内核信息的虚拟文件系统
├── root/       # root用户专属目录
├── run/        # 运行时进程数据（PID等）
├── sbin/       # 系统管理命令（fdisk、shutdown）
├── srv/        # 服务数据（网站、FTP）
├── sys/        # 内核设备树信息
├── tmp/        # 临时文件（重启清空）
├── usr/
│   ├── bin/    # 用户级应用程序
│   ├── sbin/   # 管理员高级工具
│   └── src/    # 内核源代码
└── var/
    ├── log/    # 系统日志
    └── www/    # Web服务数据
```
